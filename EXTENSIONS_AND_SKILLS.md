# Extensiones y Skills Instaladas

## 🧩 Extensiones
No hay extensiones adicionales instaladas actualmente.

## 🛠️ Agent Skills (Habilidades)

Actualmente tienes las siguientes habilidades habilitadas y descubiertas:

| Habilidad | Descripción | Ubicación |
| :--- | :--- | :--- |
| **bcra-transparencia** | Consulta la API pública del Régimen de Transparencia del BCRA (tasas, préstamos, tarjetas). | `~/.agents/skills/bcra-transparencia` |
| **expense-report-generator** | Generación de reportes de gastos profesionales desde datos o CSV. | `~/.agents/skills/expense-report-generator` |
| **find-skills** | Ayuda a descubrir e instalar nuevas habilidades para el agente. | `~/.agents/skills/find-skills` |
| **frontend-design** | Diseño de interfaces frontend de alta calidad y componentes visuales. | `~/.gemini/skills/frontend-design` |
| **gemini-api-dev** | Desarrollo con modelos Gemini, API, funciones y contenido multimodal. | `~/.agents/skills/gemini-api-dev` |
| **git-commit** | Ejecución de commits siguiendo convenciones (Conventional Commits). | `~/.agents/skills/git-commit` |
| **git-workflows** | Gestión de flujo de trabajo de Git, ramas y colaboración. | `~/.agents/skills/git-workflows` |
| **react-frontend-expert** | Patrones avanzados de implementación de React y TypeScript. | `~/.agents/skills/react-frontend-expert` |
| **release** | Creación de lanzamientos (releases), changelogs y etiquetas de Git. | `~/.agents/skills/release` |
| **visualizing-data** | Creación de visualizaciones de datos, dashboards y reportes gráficos. | `~/.agents/skills/visualizing-data` |
| **skill-creator** | Guía especializada para crear o actualizar nuevas habilidades. | *Built-in* |

---
*Nota: La lista de habilidades incluye tanto las descubiertas en el sistema del usuario como las integradas (built-in).*
